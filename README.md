# SpeechRec-2026

## Project Overview
This project focuses on binary audio classification to distinguish between "yes" and "no" responses in speech recordings. It leverages machine learning techniques to analyze and classify audio data.

## Features
- DATASET : SPEECHCOMMANDS
- Preprocessing of audio files.
- Training and evaluation of a binary classification model.


## Usage
1. Prepare your dataset with labeled "yes" and "no" audio samples.
2. Train the model using the provided scripts.
3. Use the trained model to classify new audio recordings.

## Requirements
- Python 3.x
- `pip install -r requirements.txt`
