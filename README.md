# SpeechRec-2026

Course repository for **Speech Recognition – 2026**.

## Homework Assignments

| # | Topic | Directory |
|---|-------|-----------|
| HW1 | Speech Feature Extraction (waveform, spectrogram, MFCCs) | [`HW1/`](HW1/) |
| HW2 | Automatic Speech Recognition & WER Evaluation | [`HW2/`](HW2/) |

## Quick Start

```bash
# Install all dependencies
pip install -r requirements.txt

# Open a homework notebook
jupyter notebook HW1/HW1.ipynb
jupyter notebook HW2/HW2.ipynb
```
